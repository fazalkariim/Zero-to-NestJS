# 🚀 Zero to NestJS

A structured backend learning journey starting with **TypeScript fundamentals** before moving to **NestJS**.

## 📚 Learning Roadmap

### Phase 1 — TypeScript

- TypeScript Fundamentals
- Object-Oriented Programming (OOP)
- Advanced Types
- Generics
- Modules
- Decorators
- Utility Types
- Async Programming

### Phase 2 — NestJS

- Controllers
- Providers
- Modules
- Dependency Injection
- CRUD APIs
- Validation
- Authentication
- Prisma
- PostgreSQL
- Production Best Practices

## 🎯 Goal

Build a strong TypeScript foundation first, then use those concepts to master NestJS and backend development.
